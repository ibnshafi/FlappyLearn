# Roadmap

FlappyLearn's long-term vision is to become the most approachable open-source project for watching neuroevolution learn a game from scratch.

## Phase 1: Public Launch Foundation

- Complete launch-ready README and docs.
- Add governance, support, security, and contribution files.
- Add issue templates, pull request template, CI, release workflow, and Dependabot.
- Keep the core package dependency-light and fully testable.
- Publish clear smoke-run and full-run workflows.

## Phase 2: Richer Learning Visuals

- Add best-genome network diagrams.
- Add experiment comparison pages for multiple metrics files.
- Add replay overlays for observations, action logits, and memory state.
- Add docs that connect each visual to a learning concept.
- Add gallery-ready screenshots and demo GIFs.

## Phase 3: Interactive Education

- Add a browser dashboard for training inspection.
- Add guided lessons for mutation, selection, crossover, and fitness shaping.
- Add classroom exercises and workshop-ready configs.
- Add community challenge seeds and reproducible leaderboards.

## Phase 4: Research And Community Scale

- Add parallel evaluation.
- Add checkpoint schema versioning and migration tests.
- Add algorithm extension points for alternative evolutionary strategies.
- Publish benchmark reports across seeds and configs.
- Invite community-maintained examples and curriculum packs.

## Design Principles

- Make learning visible.
- Prefer reproducibility over spectacle.
- Keep the core understandable.
- Use visual polish to clarify, not distract.
- Treat documentation as part of the user experience.
