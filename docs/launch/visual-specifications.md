# Visual Specifications

## Logo

- Canvas: 1024 x 1024.
- Shape: simple circular or rounded-square mark.
- Concept: bird silhouette crossing a pipe gap with a small neural-node motif.
- Colors: sky blue, leaf green, warm yellow, near-black text.
- Usage: `docs/assets/logo.png`, README header, docs favicon, social posts.

## Hero GIF

- Dimensions: 1280 x 720.
- Duration: 10 to 12 seconds.
- Frame rate: 24 fps.
- File size target: under 8 MB.
- Layout: left two-thirds replay canvas, right one-third metrics and generation stats.
- Text hierarchy: `FlappyLearn` as main title, `AI learns Flappy Bird from scratch` as subtitle, small labels for generation, score, and best fitness.

## Demo GIF

- Dimensions: 960 x 540.
- Duration: 8 to 10 seconds.
- Sequence: metrics curve, replay, best-genome stats.
- Best use: README opening section and social replies.

## Screenshots

Required screenshots:

- `replay.png`: best replay in a clean browser frame.
- `metrics.png`: learning curve after a meaningful run.
- `benchmark.png`: benchmark JSON summarized as a small comparison chart.
- `config.png`: compact view of the config file with highlighted trainer settings.

## Repository Social Card

- Dimensions: 1280 x 640.
- Safe margin: 80 px.
- Background: real replay frame, not abstract decoration.
- Foreground: project name, value proposition, and learning curve.
- Message: `Watch an AI evolve from zero to flying`
- Typography: bold title, medium subtitle, small feature strip.

## Thumbnail Design

- Dimensions: 1200 x 675.
- Use a replay frame with the bird approaching a pipe gap.
- Overlay a rising curve and the text `AI learns Flappy Bird`.
- Keep contrast high for small previews.
