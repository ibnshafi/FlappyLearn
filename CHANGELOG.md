# Changelog

All notable changes to FlappyLearn are documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and the project uses [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Launch-ready README, governance, support, security, contribution, and release documentation.
- MkDocs documentation structure for concepts, configuration, visualization, roadmap, and launch assets.
- GitHub issue templates, pull request template, CI workflow, release workflow, and Dependabot configuration.
- Ruff, pytest, mypy, pre-commit, EditorConfig, and Git attributes configuration.

## [0.1.0] - 2026-06-01

### Added

- Deterministic Flappy Bird simulator with seedable physics.
- Self-adaptive recurrent circuit genome with mutation, crossover, topology changes, and serialization.
- Population-based discovery trainer with elitism, tournament selection, novelty pressure, and curriculum scheduling.
- CLI commands for training, evaluation, benchmarking, replay export, metrics visualization, and profiling.
- JSON checkpointing and best-genome persistence.
- Dependency-free HTML replay and metrics artifacts.
- Smoke, default, and full training configs.
- Unit and smoke tests for core behavior.
